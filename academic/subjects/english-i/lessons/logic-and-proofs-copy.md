# Logic and Proofs - Copy

## Overview
Learning Goal: By the end of this unit, a student should be able to understand logical statements, translate real-world problems into logical form, apply rules of inference, and construct correct mathematical proofs independently.

### Introduction to Logic
Logic is the mathematical study of correct reasoning. It helps us decide whether a conclusion logically follows from given assumptions (premises).
Why Logic Matters (with examples)

 	Mathematics: Proving theorems (e.g., proving √2 is irrational)
 	Computer Science: Ensuring programs behave correctly
 	Everyday reasoning:

 	Premise: If it rains, the road is wet.
 	Premise: It rains.
 	Conclusion: The road is wet.



This reasoning is logical and valid.

### Propositions
A proposition is a declarative statement that is either true or false, but not both.
